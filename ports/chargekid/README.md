## Controls

| Button | Action |
|--|--| 
|L-stick/D-pad|Movement|
|A/B|Jump|
|X/Y |Bullet Boost|
|L1/R1|Shoot|
|Start|Select (at menu)|
|Select|Escape/Pause|
