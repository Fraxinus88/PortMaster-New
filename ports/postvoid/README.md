## Controls

| Button | Action |
|--|--| 
|D-pad/L-stick|Movement|
|R-stick|Mouse/Aim|
|L1 /L2|Slide|
|R1/R2 |Shoot/Select|
|A/B|Jump|
|X/Y|Reload|
|Start|Pause/Menu|
|Select|End Run|
