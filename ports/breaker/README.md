## Controls

| Button | Action |
|--|--| 
|Stick Left/ Left Trigger|turn BLUE and move COUNTER-CLOCKWISE|
|Stick Right/ Right Trigger|turn RED and move CLOCKWISE|
|A|Start playing/Input Initials|
|Start|Pause/Menu|
