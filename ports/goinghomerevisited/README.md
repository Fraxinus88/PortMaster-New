## Controls

| Button | Action |
|--|--| 
|DPAD-left, DPAD-right|Movement|
|LSTICK, RSTICK, R2+DPAD|Mouse|
|R1|Mouse left btn|
|Y, DPAD-down|Toggle flashlight|
|X, DPAD-up|Interact|
|A, B|Enter/Escape on puzzles|
|Start|Pause|
