## Controls

| Button | Action |
|--|--| 
|A|Accelerate|
|B|Break|
|R1|Shift UP|
|L1|Shift Down|
|DPAD| Steer |


## Compile

```shell
git clone https://github.com/ptitSeb/f1spirit.git
make
```
