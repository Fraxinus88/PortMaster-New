## Controls

| Button | Action |
|--|--| 
|D-pad/L-stick|Movement |
|A |Jump|
|B|Crouch/Down|
|Start|Enter|
|Select|Pause/Menu|
|Select + B (Hold) |Exit to Mainscreen while in Pause Menu|
