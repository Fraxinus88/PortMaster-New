|--|--| 
|DPAD| Move |
|A| Select |
|B| Back / Menu |

## Compile O.H.R.RPG.C.E 

```shell
Install FreeBASIC
git clone https://github.com/ohrrpgce/ohrrpgce.git
cd ohrrpgce
scons gfx=sdl2
```
