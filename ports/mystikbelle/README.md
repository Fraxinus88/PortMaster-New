## Controls

Cursor Keys = Move
Z = Jump A
X = Attack, Use Items B
C = Check, Talk, Inventory Y
Enter = Game Menu.
ESC (x2) = Quit Game

| Button | Action |
|--|--|
| Dpad | Move |
| Select | Back |
| A | Jump |
| B | Attack / Use Items |
| Y | Check, Talk, Inventory |
