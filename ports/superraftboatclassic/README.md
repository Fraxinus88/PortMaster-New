## Controls

| Button | Action |
|--|--| 
|D-pad/L-stick|Movement|
|R-stick|Aim|
|L shoulder|Build|
|R shoulder|Fire|
|A/B|Confirm/Back|
|Start|Pause|
|L2/R2 + Dpad|Continue from Unlock Screen|
