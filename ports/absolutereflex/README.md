## Controls

| Button | Action |
|--|--|
|DPAD/RSTICK+A/X/R1/R2(hold)|Ship movement|
|DPAD/RSTICK|Cursor movement|
|A/X/R1/R2(press)|Teleport|
|A/X/R1/R2(hold)|Fire|
|Select|Exit|
