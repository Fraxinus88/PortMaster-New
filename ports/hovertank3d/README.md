## Controls

The following instructions are for a right-facing character. 

| Button | Action |
|--|--| 
|DPAD| Move |
|A| Fire |
|B| Thrust |
|Y| 1|
|X| 2|

## Compile ## 

```bash
git clone https://github.com/Cebion/SDL_Hovertank3D
mkdir build && cd build
cmake ..
make
```
