## Controls

| Button | Action |
|--|--| 
|D-pad|movement|
|L-analog stick|movement|
|A-button|jump/start|
|X-button|fire bubbles|
|D-pad_down|chew gum|
|L-analog_down|chew gum|
|Start-button|Start|
