## Controls

| Button     | Action              |
| ---------- | ------------------- |
| LEFT/RIGHT | Horizontal movement |
| DOWN       | Fall                |
| A          | Jump                |
| B or R1    | Switch              |
| L1         | Reload              |
| START      | Pause               |
