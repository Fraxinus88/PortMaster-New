## Controls

| Button | Action |
|--|--| 
|A| Left Mouse|
|B| Right Mouse|
|X| Slow Mouse (HOLD)|
|Y| Show Help |

## Compile

```shell
git clone https://github.com/kuroneko/sais.git
cd said
mkdir build && cd build
cmake ..
make
```
