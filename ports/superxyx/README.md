## Installation

SuperXYX requires a particular release, to get it do the following.

1. Type the following on the windows explorer.exe address bar or on MacOS Safari address bar.
2. `steam://open/console`
3. Press Enter and a console window will appear within steam, type the following
4. `download_depot 654500 654501 5817820004224362769`
5. Press Enter
6. Wait for it to complete it will give you a folder where it downloaded the files these are the data files you need to use
7. Install SuperXyX in PortMaster
8. Copy all files folder from downloaded depot that steam showed to `ports/superxyx/gamedata/`
9. Launch the game
10. If you made a mistake or hear no music you go back to step 7 and try again.
