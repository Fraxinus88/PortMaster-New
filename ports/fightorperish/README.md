## Controls

| Button | Action |
|--|--| 
|Left Analogue| Move|
|Dpad| Move|
|A| Shoot|


## Compile

```shell
git clone https://gitlab.com/dulsi/fop.git
cd fop
mkdir build && cd build
cmake ..
make
```
