## Controls

| Button | Action |
|--|--| 
|DPAD/LSTICK|Move|
|RSTICK|Mouse|
|A/L1/L2|Speed-up to break walls|
|X|Next level (F) (menu option)|
|Y|Restart level (R) (menu option)|
|R1/R2|LMB|
|B/SELECT/START|Menu|
