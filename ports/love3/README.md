## Controls

| Button | Action |
|--|--| 
|D-Pad/L-Stick|Movement|
|A|Jump|
|B|Checkpoint|
|Y|Die|
|R1/R2|Slow Motion|
|Select|Reset|
|Start|Menu/Skip|
