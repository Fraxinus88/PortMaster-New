## Controls

| Button | Action |
|--|--|
| Back | Playdate Button B |
| Start | Playdate Button A |
| D-pad/Left-Analog | Movement, navigation in menus |
| A | Playdate Button A |
| B | Playdate Button B |
| X | Swap Source files (Playdate 1-bit & colored graphics) |
