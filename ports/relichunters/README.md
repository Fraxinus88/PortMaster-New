## Controls

|Button| Action |
|--|--|
| l-stick| movement  |
| l2| hold to aim|
| r-stick| aim |
| r2| fire |
| l1| grenade |
| a| hold to run/press to dash |
| b| melee |
| x| reload/pick up/dig |
| y| swap weapon |
| start| pause |
